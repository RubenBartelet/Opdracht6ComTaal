List Style Types Opdracht 6.1

  1. list-style-type: waarde;

    2. Dit attribuut kijkt naar wat er voor het genoemde woord staat. Hij kan 4 waardes aannemen:

    1. circle (rondje)
    2. square (vierkant)
    3. upper-roman (I, II, II)
    4. lower-alpha (a, b, c)

  2. list-style-image: url('meneermoorlag.png');

    3. Dit attribuut lijkt op het attribuut hiervoor maar dan met een afbeedling. Hij kan oneidig veel waardes hebben omdat je elke afbeelding kan toevoegen.

    1. Zo kan je dus een lijst maken
    2. met afbeeldingen
    3. van bijvoorbeeld iemands hoofd.
    4. Super Handig! IEDEREEN WIL DIT!

  3. list-style-position: waarde;

    4. Dit attribuut zorgt ervoor of de waarde van het attribuut binnen of buiten de tabel zit. Hij kan de waardes outside of inside hebben.

    1. Deze is erbuiten
      		2.
    2. Deze is erin
      		3.

  4. De default andere stijlen die voor elk ander element geleden als background, width, margin, height, color en padding.


Font Style Types Opdracht 6.2

  1. font: waarde, Font, Font;

    2. Dit attribuut is de default en kan je verschillende dingen invoegen als groote en type font.

  2. font-family: Font, Font, Font;

    3. Hier kan je meerdere fonts invoegen. Als een van de fonts niet werkt in de browser pakt hij een andere.

  3. font-size: waarde;

    4. Met dit attribuut kun je de grootte van de text veranderden. Dit kan je op 3 verschillende manieren doen:
    25px, 69%, large, medium en small.

  4. font-style: waarde;

    5. Dit attribuut zorgt voor de stijl van de front. Het kan de waardes normal(normaal), oblique(schuingedrukt) of italic(cursief) bevatten.

  5. font-variant: waarde;

    6. Hiermee kan je de variant van de text veranderen. Je hebt bijvoorbeeld normal(normaal) en small-caps.

  6. font-weight: waarde;

    7. Met dit attribuut kun je aangeven hoe dik je de text wilt hebben. De waardes zijn: normal(normaal), lighter(lichter), bold(dik) en een waarde met een getal hoe dik het kan zijn (met een maximum van 1000).
